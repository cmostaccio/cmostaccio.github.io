Aplicaciones

El algoritmo DFS posee varias aplicaciones, las mas importante que se pueden mencionar son

  1. Encontrar las componentes conexas de un grafo no dirigido
  2. Prueba de aciclicidad. Dado un grafo (dirigido o no dirigido) comprobar si tiene algún ciclo o no.
  3. Encontrar las componentes fuertemente conexas de un grafo dirigido.
